🚀 INSTRUCCIONES RÁPIDAS PARA NETLIFY
==========================================

1. PREPARAR ARCHIVOS
   ✅ Ya está todo listo

2. SUBIR A NETLIFY
   📁 Opción A: Arrastra toda la carpeta a netlify.com
   🔗 Opción B: Conecta tu repositorio de GitHub

3. CONFIGURACIÓN AUTOMÁTICA
   ✅ _redirects configurado
   ✅ netlify.toml configurado
   ✅ Detección automática de entorno

4. ¡LISTO!
   🌐 Tu app estará en: https://tu-app.netlify.app

ARCHIVOS IMPORTANTES:
- index.html (página de login)
- dashboard.html (página principal)
- auth-netlify.js (autenticación para Netlify)
- script.js (lógica principal)
- style.css + login.css (estilos)
- _redirects (redirecciones)
- netlify.toml (configuración)

NO SUBIR:
- node_modules/
- server.js
- datos.db
- package.json

La aplicación funcionará 100% en el frontend con localStorage!
